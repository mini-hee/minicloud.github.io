1. ~/.ssh/config 파일 생성
2. Config 설정
```
Host [호스트이름]
  HostName [private IP]
  User [user name]
  PORT [port번호]
  IdentityFile [키 위치]
```
3. 보안을 위해 권한 제한
`chmod 440 config`


※ Vagrant로 생성한 VM을 추가하려면 `vagrant ssh-config` 명령어를 통해 설정 정보를 확인할 수 있다.
